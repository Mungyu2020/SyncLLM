

How to use

1.rename_RTL.py
입력 파일: Data/1.RTL_raw/*.v
	- HDLBits solution에서 파싱한 RTL을 저장한 파일
출력 파일: Data/2.RTL/*.v 
코드 기능
	- RTL_raw의 .v 파일명을 이후에 top module이름으로 사용하기 위해 .v파일명 수정
사용법
	- (tcsh shell) python 1.rename_RTL.py

2.rename_topmodule_name.py
입력 파일: Data/2.RTL/*.v
	- HDLBits solution에서 파싱한 RTL을 저장한 파일
출력 파일: Data/3.RTL_final/*.v 
코드 기능
	- 실제 top module이름을 RTL/*.v 파일명으로 수정 (RTL/*.v 코드를 보면 top module 이름이 전부 top_module임.)
사용법
	- (tcsh shell) python 2.rename_topmodule_name.py

3.rtl_to_sog
입력 파일: Data/3.RTL_final/*.v 
출력 파일: 
	- Data/4.RTL_final/*.v 
	- Data/4.Output_graphml/*.graphml
코드 기능
	- MasterRTL code 활용하여 RTL -> SOG graph 변환
	- graphml 형태의 SOG graph 는 향후에 Sync-LLM의 EDGE diffusion model에 활용될 예정
사용법
	- (tcsh shell) python 3.rtl_to_sog.py
참고: MasterRTL의 vlg2ir/analyze.py에 graph2graphml() 함수 추가
